开机动画 bootanimation

发表于 2020-12-31 更新于 2020-01-16 分类于 烂笔头 阅读次数：

开机动画文件：bootanimation.zip
路径 ：/system/media/

bootanimation.zip文件是zip压缩文件，压缩方式要求是存储压缩，包含一个文件和两个目录：
    1.动画属性描述文件：desc.txt
    2.第一阶段动画图片目录：part0
    3.第二阶段动画图片目录：part1
desc.txt文件内容：
    480 427 30
    p 1 0 part0
    p 0 10 part1
desc.txt文件分析：
    480  427   30  
     宽   高    帧数（对应多少张图片）
      p         1              0                      part0
    标志符 循环次数 阶段切换间隔时间  对应目录名
      p         0             10                     part1 
    标志符 循环次数 阶段切换间隔时间  对应目录名
    ==================
    标志符：
        必须是: p
    循环次数：
        0 : 表示本阶段无限循环
    阶段切换间隔时间：
       &nbs
zip -Z store bootanimation.zip part0/*png part1/*png desc.txt
