LCD 配置 OUT_FACE FORMAT

发表于 2020-12-31 更新于 2020-01-16 分类于 烂笔头 阅读次数：

对于18BIT 的屏幕，这种屏幕的LVDS data format 一般如图（5）所示：

如果只使用主控的一个LCDC+RK610/RK616 的Scaler 实现双显，必须把

OUT_FACE 定义为OUT_P888_D666,LVDS_FORMAT 定义为LVDS_8BIT_2，否则在HDMI 模

式下，屏幕显示会异常。

上图说明在Y0 通道上传输的是R0R5 和G0，在Y1 通道上传输的是G1G5 和B0、

B1，在Y2 通道上传输的是B2~B5 和VSYNC、HSYNC 及DEN，在Y3 通道上传输的是各种

颜色的高两位。参考rk_screen.h 中的定义，我们知道，这个格式对应LVDS_8BIT_1。
